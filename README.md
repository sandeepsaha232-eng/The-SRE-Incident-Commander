# 🚀 SRE Fleet Commander

**The Unified Control Plane for Distributed AI-Powered Incident Remediation.**

SRE Fleet Commander is a professional-grade observability and remediation platform designed for the Meta OpenEnv Hackathon. It bridges the gap between raw system telemetry and actionable AI-driven SRE intelligence.

## 🌟 Key Features

### 1. Global Fleet Dashboard
A real-time command center that monitors every enrolled MacBook "Soldier" in your fleet.
- **Instant Status**: Healthy (Green) vs. Critical (Red) based on a hardcore **20% CPU threshold**.
- **Live Vitals**: Continuous reporting of CPU, RAM, and top process offenders.
- **Node Tracking**: Automatic hostname registration and "Last Seen" heartbeat monitoring.

### 2. AI-Powered Deep Diagnostics
Every node in the fleet has an attached **Groq-powered AI Brain** (`Llama-3.3-70b-versatile`).
- **One-Click Analysis**: Clicking any Machine ID launches a deep-dive diagnostic page.
- **SRE Recommendations**: Real-time, 2-sentence technical fixes and terminal commands generated by the AI based on live telemetry.

### 3. OpenEnv Compliance
Fully compliant with the Meta OpenEnv Hackathon requirements.
- **Standardized API**: Implements `/tasks`, `/reset`, `/step`, `/state`, and `/grader`.
- **Hybrid Architecture**: Serves both a visual dashboard for humans and a structured API for the OpenEnv validator bot.

## 🛠️ Join the Fleet (Quickstart)

Turn your local MacBook into an SRE-protected "Soldier" in under 10 seconds:

```bash
curl -sSL https://raw.githubusercontent.com/sandeepsaha232-eng/The-SRE-Incident-Commander/main/install.sh | bash
```

*This command installs `psutil` and `requests`, downloads the `fleet_agent.py`, and starts reports to the Command Center.*

## 📐 Architecture

- **The General (Central Server)**: FastAPI backend deployed on Hugging Face Spaces. Manages the fleet map and AI inference.
- **The Soldiers (Local Agents)**: Lightweight Python agents running on distributed hardware, communicating via secure JSON reports.
- **The Brain**: Groq-powered LLMs providing sub-second SRE expertise.
